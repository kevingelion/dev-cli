# Usage
<!-- usage -->
```sh-session
$ npm install -g @notarize/dev-cli
$ notarize-dev COMMAND
running command...
$ notarize-dev (-v|--version|version)
@notarize/dev-cli/0.0.1 darwin-x64 node-v14.15.0
$ notarize-dev --help [COMMAND]
USAGE
  $ notarize-dev COMMAND
...
```
